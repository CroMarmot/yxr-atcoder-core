# Install

## Clone

```bash
git clone git@github.com:CroMarmot/yxr-atcoder-core.git
cd yxr-atcoder-core
```

## Setup venv

```bash
python3 -m venv venv
```

## Enable venv

```bash
. venv/bin/activate
```

## Install dependencies

```bash
pip3 install .
pip3 install .[test]
```

## Build

```bash
python -m build
```
